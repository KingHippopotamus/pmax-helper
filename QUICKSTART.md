# 🎬 Pmax Helper - 超かんたん使い方ガイド

**プログラミング不要！ブラウザだけでAI動画が作れます**

所要時間：初回10分、2回目以降3分

---

## 📋 必要なもの

1. **Googleアカウント**（GmailやGoogle Driveを使っているアカウント）
2. **fal.ai APIキー**（無料で取得できます）
3. **動画にしたいLP（ランディングページ）のURL**

---

## 🔑 ステップ1: APIキーを取得する（初回のみ）

### fal.ai APIキーの取得方法

1. https://fal.ai/ にアクセス
2. 右上の「Sign In」をクリック
3. Googleアカウントでログイン
4. ログイン後、画面右上のアイコンをクリック→「Dashboard」を選択
5. 左メニューの「API Keys」をクリック
6. 「Create New Key」ボタンをクリック
7. 表示されたキー（`fal_...` で始まる長い文字列）をコピー
   - ⚠️ このキーは二度と表示されないので、メモ帳などに保存してください

---

## 🚀 ステップ2: Google Colabでノートブックを開く

1. 下記のリンクをクリック（新しいタブで開きます）:

   👉 **https://colab.research.google.com/github/KingHippopotamus/pmax-helper/blob/main/pmax_helper_simple.ipynb**

2. Googleアカウントでログインを求められたらログイン

3. 画面が開いたら、左上に「Pmax Helper - シンプル版」と表示されていることを確認

---

## 🔐 ステップ3: APIキーを設定する（初回のみ）

1. 画面左側のサイドバーにある **🔑 鍵マーク**（Secrets）をクリック

2. 「Add new secret」ボタンをクリック

3. **1つ目のキー**を追加:
   - Name（名前）: `FAL_KEY`
   - Value（値）: ステップ1でコピーした `fal_...` で始まるキーを貼り付け
   - 「Notebook access」のトグルを **ON** にする ✅
   - 「Save」をクリック

4. もう一度「Add new secret」ボタンをクリック

5. **2つ目のキー**を追加:
   - Name（名前）: `LAMBDA_SECRET_KEY`
   - Value（値）: `dummy_key` と入力（これは必須ですが実際には使いません）
   - 「Notebook access」のトグルを **ON** にする ✅
   - 「Save」をクリック

6. 2つのキーが登録されていることを確認

---

## ▶️ ステップ4: セルを実行する

画面には番号付きの「セル」という枠が並んでいます。上から順番に実行します。

### セル1を実行:

1. 「## 1. API キーの設定」と書かれたセルの下にある、コードが書かれたセルをクリック

2. セルの左側に表示される **▶（再生ボタン）** をクリック

3. 「✅ APIキーを設定しました」と表示されればOK
   - ❌ エラーが出た場合：ステップ3でキーが正しく設定されているか確認

### セル2を実行:

1. 「## 2. セットアップ」の下のセルをクリック

2. 左側の **▶（再生ボタン）** をクリック

3. 1-2分待つと、以下のようなメッセージが表示されます:
   ```
   ✅ セットアップ完了
   ✅ 最新コミット: xxxxxxx
   ```

### セル3を実行:

1. 「## 3. 動画生成ツール」の下のセルをクリック

2. 左側の **▶（再生ボタン）** をクリック

3. 画面に以下のような入力フォームが表示されればOK:
   ```
   🎬 Pmax Helper - AI動画生成ツール

   LP URL: [                    ]

   [🔍 ページを分析] [🎬 動画を生成]
   ```

---

## 🎥 ステップ5: 動画を生成する

### 5-1. LPを分析する

1. 「LP URL:」の入力欄に、動画にしたいランディングページのURLを貼り付け
   - 例: `https://example.com/lp/product`

2. 「🔍 ページを分析」ボタンをクリック

3. 数秒〜数十秒待つと、以下のような分析結果が表示されます:
   ```
   ✅ 分析完了！

   📊 商材名: ○○○
   🎯 ターゲット: ○○○
   💡 キャッチコピー: ○○○
   ...
   🖼️ キャラクター画像: https://...

   ✅ 動画生成の準備ができました
   ```

4. 「🖼️ キャラクター画像」にURLが表示されていることを確認
   - ❌ 「見つかりませんでした」と表示された場合：そのLPには対応する画像がありません

### 5-2. 動画を生成する

1. 「🎬 動画を生成」ボタンをクリック

2. **2〜5分待ちます**（AI処理に時間がかかります）
   - ⏳ 画面を閉じずに待ってください
   - 「🎬 AI動画生成を開始...」と表示されます

3. 完了すると、以下が表示されます:
   ```
   ✅ 動画生成完了！

   📹 動画URL: https://...

   [動画プレイヤー]

   📥 動画をダウンロード
   ```

4. 動画プレイヤーで再生して確認

5. 「📥 動画をダウンロード」リンクをクリックして保存

## 🔄 2回目以降の使い方

Google Colabを一度閉じた後、もう一度使う場合：

1. **同じタブが残っている場合**:
   - ステップ5だけ実行すればOK
   - 別のLPで生成したい場合は、URLを入れ替えて「🔍 ページを分析」から実行

2. **ブラウザを閉じた場合**:
   - ステップ2の「Google Colabでノートブックを開く」から再開
   - APIキーは保存されているので、セル1,2,3を順番に実行
   - その後、ステップ5を実行

## ❓ よくある質問

### Q1: エラーが出た場合はどうすればいい？

**A:** まず以下を確認してください：

1. APIキー（FAL_KEY）が正しく設定されているか
2. セルを上から順番に実行したか
3. インターネット接続は安定しているか

それでも解決しない場合：
- 画面上部のメニュー「ランタイム」→「セッションを再起動」
- セル1から再度実行

### Q2: 動画生成に何分かかる？

**A:** 通常2〜5分です。混雑時は10分程度かかることもあります。

### Q3: 何回でも生成できる？

**A:** fal.aiの無料枠内であれば何回でも生成できます。
無料枠を超えた場合は、fal.aiで有料プランに登録する必要があります。

### Q4: 生成した動画の商用利用はOK？

**A:** fal.aiの利用規約に従います。詳細は https://fal.ai/terms で確認してください。

### Q5: 対応しているLPの種類は？

**A:** 現在、特定のHTMLクラス名（`.wonder-cv-back-person-img`）を持つページに対応しています。
対応していないLPの場合、「キャラクター画像: 見つかりませんでした」と表示されます。

### Q6: スマホでも使える？

**A:** Google ColabはPCでの使用を推奨していますが、タブレットでも動作する場合があります。
スマホは画面サイズの問題で操作しづらい可能性があります。

## 🆘 トラブルシューティング

### 「❌ APIキーを設定しました」が表示されない

→ ステップ3でキーの「Notebook access」をONにしたか確認

### 「キャラクター画像: 見つかりませんでした」と表示される

→ そのLPには対応する画像がありません。別のLPで試してください

### セル実行時に「エラー: 401 Unauthorized」

→ FAL_KEYが間違っているか、無効になっています。fal.aiでキーを再確認

### 動画生成中にセッションが切れた

→ メニュー「ランタイム」→「セッションを再起動」して、最初から再実行

---

## 💡 使い方のコツ

1. **初回は時間に余裕を持って**
   初回セットアップは10分程度かかります

2. **複数のLPをまとめて処理する場合**
   一度セットアップしたら、ステップ5を繰り返すだけでOK

3. **生成した動画はすぐダウンロード**
   fal.aiのサーバーに保存される期間は限定的です

4. **ブラウザのタブは開いたまま**
   動画生成中（2〜5分）は、タブを閉じないでください
