# 🎬 Pmax Helper - 共有キー版 超かんたん手順書

**APIキーを1回設定するだけで、チーム全員がすぐ使えます！**

---

## 👨‍💼 管理者向け手順（初回のみ・10分）

### ステップ1: APIキーを取得する

1. https://fal.ai/ を開く

2. 右上の「**Sign In**」をクリック

3. Googleアカウントでログイン

4. ログイン後、画面右上のアイコンをクリック → 「**Dashboard**」

5. 左メニューの「**API Keys**」をクリック

6. 「**Create New Key**」ボタンをクリック

7. 表示されたキーをコピー
   ```
   fal_xxxxxxxxxxxxxxxxxxxxxxxx
   ```

   ⚠️ **重要**: このキーは二度と表示されないので、メモ帳に保存してください

---

### ステップ2: ノートブックをダウンロードする

1. https://github.com/KingHippopotamus/pmax-helper を開く

2. 「**Code**」ボタン → 「**Download ZIP**」をクリック

3. ダウンロードしたZIPファイルを解凍

4. `pmax_helper_shared_key.ipynb` ファイルを見つける

---

### ステップ3: APIキーを埋め込む

#### 方法A: テキストエディタで編集（推奨）

1. `pmax_helper_shared_key.ipynb` を**右クリック** → 「**プログラムから開く**」 → 「**メモ帳**」（Windowsの場合）または「**テキストエディット**」（Macの場合）

2. `Ctrl+F`（Macは`Cmd+F`）で検索窓を開く

3. `YOUR_FAL_KEY_HERE` を検索

4. 見つかったら、その部分を削除して、ステップ1でコピーしたAPIキーを貼り付け

   **変更前:**
   ```
   FAL_KEY = "YOUR_FAL_KEY_HERE"
   ```

   **変更後:**
   ```
   FAL_KEY = "fal_xxxxxxxxxxxxxxxxxxxxxxxx"
   ```

5. ファイルを保存（`Ctrl+S` または `Cmd+S`）

6. ファイル名を変更（例: `pmax_helper_shared_key_company.ipynb`）
   - こうすることで、誤って公開するのを防げます

#### 方法B: Google Colabで直接編集

1. https://colab.research.google.com/ を開く

2. 「**ファイル**」 → 「**ノートブックをアップロード**」

3. `pmax_helper_shared_key.ipynb` を選択

4. セル1（一番上のコードセル）をクリック

5. `YOUR_FAL_KEY_HERE` を削除して、APIキーを貼り付け

6. 「**ファイル**」 → 「**ドライブにコピーを保存**」

7. ファイル名を変更（例: `pmax_helper_shared_key_company`）

---

## 👥 メンバー向け手順（毎回・3分）

### パターンA: Google Driveから開く場合

#### 初回のみ

1. 管理者から共有されたGoogle Driveのリンクを開く

2. ファイル名「**pmax_helper_shared_key_company.ipynb**」（または類似の名前）をダブルクリック

3. 画面上部に「**アプリで開く**」と表示されたら、「**Google Colaboratory**」を選択
   - 表示されない場合: 「接続済みのアプリ」→「アプリを追加」→「Colaboratory」を検索してインストール

4. Google Colabの画面が開く

#### 毎回の使い方

1. セル1（「## 1. APIキーの設定」の下）の左側にある **▶（再生ボタン）** をクリック

2. 「✅ APIキーを設定しました（共有キー使用中）」と表示されるまで待つ

3. セル2（「## 2. セットアップ」の下）の **▶（再生ボタン）** をクリック
   - 1-2分待つと「✅ セットアップ完了」と表示される

4. セル3（「## 3. 動画生成ツール」の下）の **▶（再生ボタン）** をクリック

5. 画面に入力フォームが表示される:
   ```
   🎬 Pmax Helper - AI動画生成ツール（共有キー版）

   LP URL: [                    ]

   [🔍 ページを分析] [🎬 動画を生成]
   ```

6. 「**LP URL:**」にランディングページのURLを貼り付け
   - 例: `https://lp.example.com/product`

7. 「**🔍 ページを分析**」ボタンをクリック

8. 10-30秒待つと、分析結果が表示される:
   ```
   ✅ 分析完了！

   📊 商材名: ○○○
   🎯 ターゲット: ○○○
   💡 キャッチコピー: ○○○
   ...
   🖼️ キャラクター画像: https://...

   ✅ 動画生成の準備ができました
   ```

9. 「**🎬 動画を生成**」ボタンをクリック

10. **2〜5分待つ**（画面を閉じないでください）

11. 完成すると、動画が表示される:
    - 再生して確認
    - 「**📥 動画をダウンロード**」リンクをクリックして保存

12. **完了！**

## ❓ よくある質問

### Q1: 「✅ APIキーを設定しました」が表示されない

**A:** セル1の左にある▶ボタンを押し忘れていませんか？

### Q2: 「キャラクター画像: 見つかりませんでした」と表示される

**A:** そのLPには対応する画像がありません。別のLPで試してください。

### Q3: 動画生成中に何分待てばいい？

**A:** 通常2〜5分です。混雑時は10分程度かかることもあります。

### Q4: 「エラー: 401 Unauthorized」と表示される

**A:** APIキーが間違っているか、無効になっています。管理者に連絡してください。

### Q5: 2回目以降も毎回セルを実行する必要がある？

**A:** はい。Colabを閉じると設定がリセットされるため、毎回セル1, 2, 3を実行してください。

### Q6: 同じLPで何回も生成できる？

**A:** できます。ただし、fal.aiの無料枠を超えると課金されます（管理者のアカウントに）。

### Q7: 生成した動画の保存期限は？

**A:** fal.aiのサーバーに保存される期間は限定的です。すぐにダウンロードしてください。

### Q8: スマホでも使える？

**A:** Google Colabは基本的にPC推奨です。スマホだと画面が小さくて操作しづらい可能性があります。
